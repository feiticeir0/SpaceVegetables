# Space Vegetables Client

**Space Vegetables Client** 



The Space Vegetable Client is the Raspberry PI 4 2GB with the Pimoroni's Enviro HAT. 

	* Raspberry PI 4 2GB
	* Pimoroni's Enviro HAT
	* SHT21 sensor
	* Arduino Pro Micro (connected to the USB port)
		* TDS sensor from DFRobot
